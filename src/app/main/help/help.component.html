In order to correctly set the application up, please make sure to first create a store.

Once a store has been created, floorplans and the servers that will be working in the restaurant should be created next.

Create stores, floorplans, and servers on the 'Manager' page.

Reservations can be created on the 'Reservation' page.

Once everything has been created, interact with the floorplans on the 'Dashboard' page, where guests can be seated at tables in a live view of the restaurant.
